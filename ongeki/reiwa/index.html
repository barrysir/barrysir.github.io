オンゲキ ベスト枠画像ジェネレーター NEW

This is a copy of reiwa.f5.si: オンゲキ ベスト枠画像ジェネレーター made to generate images of your best/new frames from Kamaitachi using a script I wrote.

This mirror was made in preparation of them updating to Ongeki Refresh's new rating system.

The website was modified slightly to make the Kamaitachi process a bit smoother but is otherwise left original.

I've also added omnimix data so there won't be jacket errors hopefully.

About this tool

This is a tool that creates images of the best/new rated songs in your オンゲキ rating.

お問い合わせ

本ツールに関するお問い合わせは、こちらのGoogleフォームまたは作者X(旧twitter)アカウントにお願いします。（※XのDMは気づき辛いので可能であればリプライや他の手段を用いていただけると幸いです）

注意事項

  • プレミアムコースに登録していないユーザーは本ツールを利用できません。
  • ユーザーID入力されていない場合、本ツールは動作しません。
  • OngekiScoreLogがメンテナンスを行っているなどの事情で、本ツールが使用できないことがあります。

仕様

  • 『プレビューを表示する』にチェックが入っていなくても一時的にプレビューと同じ描画領域が表示されます。
    一時的に表示が崩れることがありますが、使用する上での問題はありません。
  • 「NEW!!」エンブレムの基準になる更新日は、テクニカルスコア以外（バトルスコアなど）の更新でも登録されます。
